底层架构

三引擎驱动

每个 Agent 不只是一个提示词。三个专业系统分别处理推理、记忆和检索 —— 让 Agent 保持智能而不会撑爆上下文窗口。

🧠

Brain

主力 LLM 负责任务执行、推理和工具调用。它接收 Agent 的人设、由 Context Engine 检索的相关记忆以及一套可用工具 —— 然后开始工作。

💜

Cerebellum

专门负责记忆整合的轻量 LLM。每次任务完成后，它决定哪些值得记住、哪些该遗忘、如何组织知识 —— 一切由认知遗忘曲线管理。

🔍

Context Engine

智能检索系统，根据当前任务拉取最相关的记忆。保持上下文精简且 token 高效 —— Agent 只看到重要的信息，永远不会被无关历史淹没。
